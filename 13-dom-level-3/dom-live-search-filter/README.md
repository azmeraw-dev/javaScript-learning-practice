# Live Search Filter (DOM Level 3)

This project filters a list of items in real-time as the user types.

## Concepts Practiced
- DOM selection
- `keyup` event
- Real-time user input
- String comparison using `includes()`
- Dynamic UI updates

## How It Works
- User types in the input field
- JavaScript compares input text with list items
- Matching items remain visible
- Non-matching items are hidden

## Skills Gained
This is a common feature in:
- Search bars
- Product lists
- Admin dashboards
